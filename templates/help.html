Data Management

Managing Your Datasets

Learn how to upload, organize, and maintain your epidemic data.

Our system supports the following file formats:

  • CSV - Comma-separated values file
  • JSON - JavaScript Object Notation file
  • XLSX/XLS - Microsoft Excel spreadsheet

Required columns for all datasets:

  • Date - Date of the record (YYYY-MM-DD format)
  • Location - Geographical location (city, region, country)
  • Cases - Number of confirmed cases

Recommended optional columns:

  • Deaths - Number of deaths
  • Recovered - Number of recovered cases
  • Latitude - Geographical latitude (for map visualization)
  • Longitude - Geographical longitude (for map visualization)

To upload a new dataset:

  1. Go to the Upload page
  2. Drag and drop your file or click to browse
  3. Enter a name and optional description for your dataset
  4. Click "Upload Dataset" to process your file

Note: Large files may take some time to process. Please be patient during the upload.

From your dashboard, you can manage all your datasets:

  • Visualize - View interactive visualizations of your data
  • Export - Download your data in various formats
  • Delete - Remove datasets you no longer need

To share a dataset with other users:

  1. Go to the dataset's visualization page
  2. Look for the "Share" button
  3. Enter the email address of the user you want to share with
  4. Select permission level (view or edit)
  5. Click "Share" to send the invitation
Visualization Guide

Understanding Visualizations

Learn how to create and interpret different types of visualizations.

Geographic Map

The map visualization shows your data points on a world map:

  • Each circle represents a location
  • Circle size indicates the value (cases, deaths, etc.)
  • Color intensity also corresponds to the value
  • Click on circles to see detailed information

Tip: Use the dropdown menu to switch between different metrics (cases, deaths, recovered).

Trend Analysis

The trend analysis shows how your data changes over time:

  • X-axis represents dates
  • Y-axis shows the values (cases, deaths, etc.)
  • Multiple metrics can be displayed simultaneously
  • Toggle between line and bar charts

Tip: Hover over data points to see exact values for specific dates.

Pro Tip

For better geographical visualizations, make sure your dataset includes latitude and longitude coordinates. If these are missing, our system will attempt to geocode locations based on names, but accuracy may vary.
